数据展示

数据项 说明
用户名称 B站用户名
粉丝数 当前粉丝总数
关注数 关注的用户数量
获赞数 获得的点赞总数
播放量 视频总播放量
个人签名 用户设置的个性签名

⚠️ 已知问题

  • 由于哔哩哔哩的等级是svg图片，用户等级信息可能无法正常获取
  • 部分用户数据可能因为隐私设置而无法显示

参数选项

你可以通过添加以下参数来自定义显示结果：

参数 值 说明
theme light / dark 设置卡片的主题风格
cn true / false 是否使用力扣中国区数据（默认false，使用美区数据）
cacheSeconds 数字 配置缓存的秒数（默认120秒）

主题选择

LeetCode区域选择 US CN

使用中国区LeetCode数据：

![](BASE_URL/leetcode/用户名?cn=true)

组合多个参数

使用暗色主题和中国区数据：

![](BASE_URL/leetcode/用户名?theme=dark&cn=true)

使用方法

将上述代码添加到你的GitHub profile README.md文件中，将"用户名"或"userId"替换为你的相应平台的用户名或ID。
