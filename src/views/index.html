GitHub & LeetCode Stats

GitHub 访问统计

在你的GitHub个人资料中添加访问计数器：

![](${BASE_URL}/github/用户名)

LeetCode 解题统计

在你的GitHub个人资料中添加LeetCode统计：

![](${BASE_URL}/leetcode/用户名)

参数选项

你可以通过添加以下参数来自定义显示结果：

主题选择

使用暗色主题：

![](${BASE_URL}/leetcode/用户名?theme=dark)

LeetCode区域选择 US CN

使用中国区LeetCode数据：

![](${BASE_URL}/leetcode/用户名?cn=true)

组合多个参数

使用暗色主题和中国区数据：

![](${BASE_URL}/leetcode/用户名?theme=dark&cn=true)
